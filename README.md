# AR Food Menu

This project hosts 3D food models that can be viewed in Augmented Reality (AR) using a QR code.

## Features

- View 3D food models
- Works on Android and iPhone
- QR code support
- Built using HTML and model-viewer

## Project Structure

```
index.html
models/
```

## Technologies

- HTML
- model-viewer
- GLB
- USDZ
- GitHub Pages
